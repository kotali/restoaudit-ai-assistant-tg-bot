# Telegram RestoAudit AI RAG ERP Bot

## Установка
1. Создай бота в @BotFather.
2. Добавь .env.
3. `pip install -r requirements.txt`
4. Деплой на Vercel: Импорт репозитория → Deploy.

## RAG Setup
- Кидай файлы в /data (PDF, TXT).
- Бот авто-обновляет из чатов в data/chats.txt.

## Документация
- MkDocs: `mkdocs serve`
- Notion: Авто-обновляется при деплое.
